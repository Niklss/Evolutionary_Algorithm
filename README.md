# Evolutionary_Algorithm
Installation:

1) Clone repository.

2) Run this commands

	sudo apt-get install python3.6

	sudo apt-get install pip3

	pip3 install numpy

	pip3 install os

	pip3 install pillow
	
	pip3 install multiprocessing

3) Add to the project folder picture 512x512 with name ‘input.jpg’.

4) Run main.py.
